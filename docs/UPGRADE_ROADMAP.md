# 升級路線圖

## v2 已完成

- 第一階段 MVP 全功能。
- 第二階段可用原型：筆順描紅、語音朗讀、練習紙列印/另存 PDF、班級競賽、家長週報、AI 例句、AI 錯題推薦、多語音標、學校多班級、訂閱方案。
- 第三階段預留模組：OCR 批改、手寫識別記錄、閱讀理解、小作文、出版社合作、LMS 串接。

## v3 建議技術升級

### 後端

- Node.js/NestJS 或 Python/FastAPI。
- PostgreSQL 作主資料庫。
- Redis 處理競賽排行榜和任務隊列。
- S3/R2/OSS 儲存圖片、音頻、作業照片和 PDF。

### AI

- 例句生成：`/api/ai/sentence`
- 錯題推薦：`/api/ai/recommend`
- 小作文評語：`/api/ai/writing-feedback`
- 需要內容安全策略：年齡分級、敏感內容過濾、老師審核。

### OCR / 手寫識別

- OCR：作業圖片 → 版面定位 → 文字/框線識別 → 老師校正。
- 手寫：Canvas 軌跡點 → 筆畫順序/方向/位置判斷 → 即時回饋。

### LMS / 校園版

- SSO：Google/Microsoft/SAML。
- Roster Sync：同步學校、班級、老師、學生。
- Grade Passback：把作業分數回傳 LMS。
- LTI 1.3：作為外部工具嵌入學校平台。

## v4 商業化

- 學校訂閱、班級版、家庭版。
- 教材出版社授權內容。
- 可列印作業包和課堂互動白板模式。
- 數據看板：學校、年級、班級、學生多層級分析。
